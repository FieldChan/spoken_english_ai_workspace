# Daily Practice Logs

Create one Markdown file per practice day.

Recommended file names:

- `YYYY-MM-DD.md` for daily logs
- `YYYY-MM-DD-weekly-review.md` for weekly reviews
- `YYYY-MM-monthly-review.md` for monthly reviews

Use `templates/daily-practice-log.md` for daily logs.

Every completed log should be committed and pushed to GitHub on the same day.
